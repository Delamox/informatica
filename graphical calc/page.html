Extremely innefficient graphing calculator using a table. careful with parabolas and big field sizes, i was too lazy to implement a break when the value of y exceeds the size of the table, making it loop millions of times more than it should.

example trigonometric: >  80*Math.cos((2*Math.PI*x)/100)+75+.5*x  
y = 80cos( ( 2πx ) / 100 ) + 75 + 0.5x

example quadratic: >  0.1*x**2+x*-10+260  
y = 0.1x² + -10x + 260

example basic: >  2*x  
y = 2x

example line through middle: >  Xaxis/2  
y = Xaxis/2
